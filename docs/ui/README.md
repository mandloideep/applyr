# Applyr UI Design - ASCII Mockups

## Overview

This folder contains ASCII mockups for the Applyr application UI.

## Structure

```
ui/
├── README.md           ← You are here
├── extension.md        ← Chrome Extension (sidebar style)
└── dashboard/
    ├── home-and-nav.md ← Navigation + Dashboard Home
    ├── jobs.md         ← Jobs Pipeline (Kanban/Table/List) + Job Detail
    ├── learning-social.md ← Learning History + Social Media Manager
    ├── email-interview.md ← Email Inbox + Interview Prep
    └── other.md        ← Applications, Resumes, Analytics, Settings
```

## Design Principles

- Sidebar navigation for both extension and dashboard
- Light and dark theme support
- Clean, minimal interface
- Action-focused workflows
